## ✨ 主要功能

- 🛠️ **多种实用工具** - 包含计算器、单位转换、二维码生成等日常实用工具
- 📱 **多平台支持** - 支持Windows、Android、微信小程序和支付宝小程序
- ⚡ **轻量高效** - 占用资源少，运行流畅，启动快速
- 🎨 **简洁美观** - 现代化UI设计，操作直观，视觉舒适

## 📖 使用说明

1. **工具选择** - 主界面展示所有可用工具，点击即可进入
